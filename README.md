## ✨ Features

- 🔬 **Health Risk Prediction** — Low, Moderate, or High risk classification
- 💡 **Personalised Recommendations** — Up to 5 actionable tips per prediction
- ⚖️ **Auto BMI Calculation** — Computed live from weight and height
- 📋 **Prediction History** — View all past predictions in session
- 📊 **Session Stats** — Aggregate confidence, BMI, and risk distribution
- 🌐 **REST API** — Clean JSON endpoints for all features
- 🎨 **Beautiful UI** — Responsive, modern frontend with animated results

## 🧬 Features Used (18 Total)

| Category | Features |
|---|---|
| Body Stats | Age, Gender, Weight, Height, BMI |
| Lifestyle | Activity Level, Sleep Hours, Water Intake |
| Diet — Healthy | Vegetables, Fruits, Protein (days/week) |
| Diet — Unhealthy | Junk Food, Soda, Late-Night Eating, Skip Breakfast |
| Engineered | BMI Category, Healthy Score, Unhealthy Score |

## 📡 API Endpoints

| Method | Route | Description |
|---|---|---|
| GET | `/` | Web dashboard UI |
| POST | `/predict` | Predict health risk + get recommendations |
| POST | `/bmi` | Calculate BMI and category |
| GET | `/history` | Last 20 predictions (session) |
| GET | `/stats` | Aggregate session statistics |
| GET | `/health` | API health check |

### Example `/predict` Request

```bash
curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d '{
    "age": 25,
    "gender": "Male",
```

## 🔬 How It Works

1. **User fills** their health profile — body stats, lifestyle, eating habits
2. **BMI is auto-calculated** from weight and height
3. **Three engineered features** are derived: BMI category, Healthy Score, Unhealthy Score
4. **18 features are scaled** using StandardScaler
5. **Logistic Regression** predicts risk class with probability scores
6. **Recommendations** are generated based on the user's specific inputs
7. **Result is returned** as JSON and rendered beautifully in the UI

---

## 📊 Dataset

- **5,000 synthetic samples** generated with realistic distributions
- **Stratified train/test split** (80/20)
- **5-fold cross-validation** used for model selection
- Risk labels derived from a weighted combination of lifestyle factors
